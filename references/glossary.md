# RTL Agent Glossary / 术语表

在 executable rules 和 specs 中，English terms 应保持一致。中文说明用于帮助 human review，不替代关键英文术语。

| English term | 中文说明 |
|---|---|
| synthesizable | 可综合 |
| valid-ready handshake | valid-ready 握手协议 |
| backpressure | 反压 |
| signedness | 有符号/无符号语义 |
| bit-width | 位宽 |
| self-checking testbench | 自检查测试平台 |
| scoreboard | 记分板/结果比较器 |
| golden model | 参考模型 |
| assertion / SVA | 断言 |
| elaboration | elaboration / 展开 |
| lint | 静态检查 |
| waveform | 波形 |
| reset polarity | 复位极性 |
| synchronous reset | 同步复位 |
| asynchronous reset | 异步复位 |
| transaction loss | transaction 丢失 |
| duplicate transaction | transaction 重复 |
| data stability | 数据稳定性 |
